Git Gui
	Pasos para la utilizacion del git gui con la cuenta de Git Hub
	-Crear un nuevo repositorio: seleciono la carpeta, ej: clase1, hacer el stage changed luego esscribir un texto antes del commit inicial, commit .
	-Despues ir a opcion remote y luego add.
    -Crear el respositorio en la cuenta de github, en add poner el nombre "origin".
    -En location la direccion de la cuenta en donde se encuentra el repositorio creado ej: "https://github.com/miguelangelvidal/nuevogit.git".
	-La opcion fetch immediately tiene que estar activada las otras dos no es necesario. 
	-luego volver a la opcion remote y opciones push y otra vez push y escribir el usuario de github luego la contraseña 
	-el proyecto, creato como repositorio se guarda en la carpeta creada en github, por ultimo si quiero descargar el proyecto subido, descargar el zip.

	En caso de eliminar o modificar 
	-Si un archivo de elimina tambien puede ser eliminado del repositorio github... 
	-Si se modifica pasa lo mismo.
